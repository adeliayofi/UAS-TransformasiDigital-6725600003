Snapmoo Photobooth Portable Tegal

## Deskripsi Project

Snapmoo Photobooth Portable Tegal merupakan website UMKM yang menyediakan layanan photobooth portable untuk berbagai acara seperti wisuda, ulang tahun, gathering, seminar, event kampus, dan acara komunitas.
Website ini dibuat sebagai bagian dari proyek UAS Mata Kuliah Transformasi Digital dengan tujuan membantu proses digitalisasi usaha melalui media promosi online dan sistem pemesanan yang lebih mudah, cepat, dan efisien.
Melalui website ini pelanggan dapat melihat informasi usaha, galeri hasil foto, paket layanan, melakukan booking, serta menghubungi pihak Snapmoo secara langsung.



## Fitur Website

- Halaman Home
- Halaman About
- Halaman Layanan
